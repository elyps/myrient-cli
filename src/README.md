# Quellverzeichnis (src)

Dieses Verzeichnis enthält den primären Quellcode für das `myrient-cli` Skript.

- `myrient-cli.sh`: Das Hauptskript, das die gesamte Logik für die Interaktion mit der Myrient-Website, die Menüführung und die Download-Verwaltung enthält.
- `VERSION`: Eine einfache Textdatei, die die aktuelle Versionsnummer des Skripts speichert.
