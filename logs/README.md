# Log-Verzeichnis (logs)

Dieses Verzeichnis enthält Protokolldateien, die vom Skript generiert werden.

- `.download_history`: Eine Verlaufsdatei, die alle erfolgreich abgeschlossenen Downloads mit Zeitstempel, Konsole und Dateiname protokolliert.
- `*.log`: Temporäre Log-Dateien, die von `wget` für jeden einzelnen Hintergrund-Download erstellt werden, um den Fortschritt zu verfolgen.
