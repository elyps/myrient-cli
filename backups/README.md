# Backup-Verzeichnis (backups)

Dieses Verzeichnis dient zur Speicherung von Backups.

- **Konfigurations-Backups**: Wenn Sie die Konfiguration über das Skript sichern, werden die `.bak`-Dateien hier abgelegt.
- **Projekt-Backups**: Das `manage.sh`-Skript kann vollständige ZIP-Backups des gesamten Projektverzeichnisses erstellen, die ebenfalls hier gespeichert werden.
